# Identify-and-remove-suspicios-browser-extention
### Steps Taken:
1. Opened about:addons to review installed extensions.
2. Checked permissions and reviews for each.
3. Removed “Old PDF Converter” (unverified, high permissions).
4. Restarted Floorp browser and checked performance.

- Quick Translate (unused)

Result: Browser performance improved and no suspicious extensions remain.
